Variables en java

Todo lenguaje de programacion utiliza variables, estas son espacios de memorias reservados para guardar informacion. Esta informacion tiene asociada un tipo de dato. Es decir, cada variable guarda exclusivamente un tipo de dato.

En Java existen distintos tipos de variables que pueden agruparse en dos categorias principales:

  1. Primitivos: Proveidos por el compilador
  2. Abstractos: Creados por el programador

Definicion de Variables:

Para definir una variable en Java primero se coloca el tipo de dato seguido del nombre de la variable; la linea debe cerrar con un punto y coma ";":

> double numero;


Se pueden definir a la vez varias variables de un mismo tipo separando sus nombre con una coma ",":

> boolean unaVariable, otraVariable, terceraVariable;


Tambien se puede inicializar una variable al momento de su definicion:

> int numero1 = 10;
